# Dotfiles

My personal configuration files, optimized for a seamless development experience on **macOS** and **Linux**.

## Documentation

- [Setup Guide](setup.md) - Detailed bootstrapping instructions.

## Quick Start

Install required tools first:

- `git`
- `curl`
- `jq`

Then:

```bash
git clone https://github.com/pablon/dotfiles.git ~/dotfiles && cd ~/dotfiles/
./setup.sh
```
